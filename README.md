# # 🏠 個人房貸試算器報告

## 📌 專案簡介

本專案為使用 C# Windows Forms 開發之個人房貸試算器，提供使用者一個簡單直觀的介面，輸入房貸相關資訊後，即可快速計算每月還款金額、總利息及總還款金額等重要資訊。

系統支援自備款比例、利率、貸款年限與寬限期設定，並透過本息平均攤還法進行計算。所有結果皆經過格式化處理，提升可讀性與使用體驗。

本專案主要練習 GUI 設計、事件處理、輸入驗證與金融計算邏輯，並結合 GitHub 進行版本控制管理。

## 一、執行說明

本系統為一個以 C# Windows Forms 實作之個人房貸試算器，提供使用者輸入房貸相關資訊後，即可快速計算每月還款金額與總利息等重要資訊。

---

### （一）輸入項目說明

使用者需輸入以下資訊：

- **房屋總價（元）**  
  輸入房屋的總價格，單位為新台幣，例如：15000000。

- **自備款比例（%）**  
  輸入自備款所佔比例，例如：20 表示自備款為 20%。

- **年利率（%）**  
  輸入銀行提供的貸款年利率，例如：2.15。

- **貸款年限（年）**  
  輸入貸款的總年數，例如：30 年。

- **寬限期（年，可填 0）**  
  若有寬限期，可輸入年數；若無則填 0。寬限期內僅需繳利息，不需償還本金。

---

### （二）操作流程

1. 輸入上述所有欄位資料  
2. 點擊「開始計算」按鈕  
3. 系統會即時顯示試算結果  
4. 若需重新輸入，可點擊「清除」按鈕恢復預設值  

---

### （三）輸出結果說明

系統會顯示以下結果：

- **貸款總金額**  
  房屋總價扣除自備款後的實際貸款金額。

- **每月應繳金額**  
  每個月需支付的金額（包含本金與利息，或寬限期內僅利息）。

- **首期利息**  
  第一期所需支付的利息。

- **首期本金**  
  第一期實際償還的本金（若在寬限期內則為 0）。

- **總利息支出**  
  整個貸款期間所支付的利息總和。

- **總還款金額**  
  貸款本金加上總利息的總支出。

---

所有數值皆以千分位與小數點後兩位進行格式化，提升可讀性。
